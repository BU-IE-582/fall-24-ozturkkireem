Data Preprocessing

Suspended and stopped matches are removed from the data and missing values are handled my taking the mean of other available rows corresponding to the same match(fixture_id) due to high missing value rate. After filling NAs with the mean, some missing values still exist, so their values taken as zero. 38 rows do not have current state column, there are excluded from the data.

Far right path (node 30) is logical, if current state is tie and home time is succesful at attacking but away team is making interceptions succesfully and getting throwings, match is likely to be won by the away team. However node 15 might be indicating that probabilities(or odds) given by bookmakers is contradicting with the result. While current state is away team winning and according to odds away team is more likely to win but fitting shows that result is home team winning. But with increasing the tree depth, our model can learn more reasonable rules which may result in better model performance. Due to high volume of data and feature number, this tree is too shallow.
